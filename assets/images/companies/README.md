# Company Logos

This directory contains logos for the companies listed in your experience section.

## Required Logos:

1. **rakuten-logo.png** - Rakuten Group logo
2. **takeme-logo.png** - TakeMe株式会社 logo
3. **delhivery-logo.png** - Delhivery Ltd. logo

## Logo Resources:

### Rakuten Group

## Specifications:

- **Format**: PNG (with transparent background preferred)
- **Size**: 48x48px to 200x200px (will be displayed at 48x48px)
- **File naming**: Use lowercase with hyphens (e.g., `rakuten-logo.png`)

## How to Add Logos:

1. Download the logo from one of the sources above
2. Save it in this directory with the correct filename
3. The portfolio will automatically display the logo next to the company name
4. If a logo fails to load, it will be hidden automatically
